Examen Psicológico UAM
Sistema web para la automatización de los Test de Razonamiento Forma A y Forma B de la Batería Factorial de Aptitudes (BFA).

Descripción:
Este proyecto tiene como objetivo digitalizar el proceso manual de aplicación, registro y calificación de pruebas psicológicas de razonamiento. El sistema permite registrar aplicantes, administrar cuestionarios, cargar preguntas, definir opciones de respuesta, registrar respuestas del sujeto y generar resultados finales.

Funcionalidades:
- Registro de aplicantes.
- Administración de cuestionarios Forma A y Forma B.
- Registro de preguntas y opciones de respuesta.
- Identificación de respuestas correctas.
- Aplicación de prueba con estado: Asignado, En Progreso, Finalizado y Calificado.
- Cálculo de respuestas correctas, incorrectas y sin responder.
- Generación de resultado final.
- Prototipo HTML para responder el test con temporizador.

Proyecto desarrollado para la asignatura de Programación Orientada a Objetos
